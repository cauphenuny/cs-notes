## 基本组成
存储器：
- MAR: Address Register
- MDR: Data Register
运算器：
- ACC: Accumulator
- MQ: Multiplier-Quotient Register
- X

|     | ACC    | MQ      | X   |
| --- | ------ | ------- | --- |
| 加法  | 被加数/和  |         | 加数  |
| 减法  | 被减数/差  |         | 减数  |
| 乘法  | 积高位    | 被乘数/积地位 | 乘数  |
| 除法  | 被除数/余数 | 商       | 除数  |

## 技术指标

字长
主频
MIPS: 每秒执行百万条指令
CPI: 每条指令所需周期数
FLOPS: 每秒浮点运算次数

$$
\text{Clock Cycles}=\sum_{i=1}^n (\text{CPI}_{i}\times\text{Instruction Count}_{i})
$$
$$
\text{CPI}=\dfrac{\text{Clock Cycles}}{\text{Instruction Count}}=\sum_{i} \left( \text{CPI}_{i} \times \dfrac{\text{IC}_{i}}{\text{IC}} \right)
$$

IPC: 每个时钟周期平均执行的指令数，现代CPU中通常大于1（流水线、超标量）
存储容量：单元个数 $\times$ 字长
