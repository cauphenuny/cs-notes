## 移位

`SHL/SHR/SAL/SAR` 逻辑/算数 左/右移
`SHLD/SHRD` 双精度左/右移

`SHL/SAL dest, count`：将dest左移count位，CF保存最后移出的一位，右侧补0
	- `count=0`：所有标志位不受影响
	- `count>0`：CF: 最后移出一位，SF、ZF、PF根据移位结果设置，AF未定义，OF = count == 1? {移动后最高位值不变} : {未定义}

`SHR`：将dest向右移动count位，CF保存最后移出的一位，左侧高位补count个0
标志位：
- 当count = 0时，所有标志位不受影响
- 当count > 0时，CF位等于最后移出的一位，SF、ZF、PF根据移位结果设置，AF未定义
- 当count = 1时，OF = 移位前dest的最高位；当count > 1时，OF未定义

`SAR`：补dest最高位

双精度左右移：增加一个操作数，`SHLD/SHRD dest, src, count`
补充的内容由src决定

## 乘除

`MUL/DIV`：无符号数
`MUL src`: src $\times$ AL/AX/EAX $\to$ AX/DX:AX/EDX:EAX，冒号前是高位
`DIV src`: (dest)AX/DX:AX/EDX:EAX $\div$ src $\to$ AH:AL/DX:AX/EDX:EAX，商在低位寄存器

`IMUL`
- `IMUL src`
- `IMUL dest, src`
-  IMUL `dest, src1, src2`

IDIV
注意事项
- 8位除法被除数是16位，16位除法被除数是32位，32位除法被除数是64位
- 当不能整除时，商的截取趋向0
  如-5 ÷3，商=-1， 余数=-2

## 符号扩展

`CBW(intel) CBTW(AT&T)`：字节扩展成字，隐含操作数$AL\to AX$
`CWDE(intel) CWTL(AT&T)` : 字扩展成双字 `E` 的意思是扩展至 `EAX` 寄存器 $AX\to EAX$
`CWD / CWTD`: 字扩展成双字，$AX\to DX:AX$

## 装载地址

不取址，而是把有效地址存起来

`(intel) LEA dest, src`

e.g. `lea  (%ebx, %esi, 8), %edi`: %edi=%ebx+8*%esi

## 设置操作数

`SETcc dest`，cc是待定条件码，若条件码成立则dest=1反之=0
