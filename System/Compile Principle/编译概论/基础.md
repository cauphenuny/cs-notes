
## 作用域

```
名字       内存位置（变量）        值
 |           |       |          |
 +->-环境->--+        +->-状态->-+
```

静态作用域：仅凭代码就能确定名字所指的是哪个位置
动态作用域：需要考虑实时调用链，从下至上选择最近的声明

```c
int a;
void f() {
	int a;
	g(a);
	h();
}
void g(int b) {
	... b ...;
}
void h() {
	... a ...;
	
}
```

考虑 $h$ 中的  `a`:
Static: refers to global `a`;
Dynamic: refers to `f::a`.
